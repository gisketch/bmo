## Requirements
### Requirement: CRT Visual Effect
The system SHALL display a CRT-style visual effect over the BMO screen to simulate an old monitor.

#### Scenario: Screen rendering
- **WHEN** the BMO screen is rendered
- **THEN** it displays scanlines, a subtle vignette, a slight flicker, and a subtle warp/curvature effect without obstructing the content
